# Apex Legends Rank Tracker Bot

Notifier change of rank every 1 minutes

## Installation & Run

Install all libraries

1. discord.js@11.5.1

2. node-json-db@1.1.0

3. node-schedule@1.3.2

4. request@2.88.2

### From Source

```
node index.js
```
